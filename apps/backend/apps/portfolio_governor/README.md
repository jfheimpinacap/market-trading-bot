# Portfolio Governor

Capa de gobernanza agregada de cartera (paper/demo only) para:

- Construir snapshots auditables de exposición agregada.
- Detectar señales de régimen (concentración, drawdown, capital tight, etc.).
- Emitir decisiones de throttling global para nuevas entradas.
- Integrarse aguas arriba con opportunity supervisor, allocation y mission control.

No reemplaza `risk_agent` ni `position_manager`; complementa a nivel portafolio.
