# Promotion Committee

Formal stack change governance layer for paper/demo environments.

## Scope
- Consolidates evidence from champion/challenger, readiness, profile governance, portfolio governor, model governance and memory precedents.
- Produces explicit recommendation codes.
- Keeps auditable review runs and decision logs.
- Supports explicit manual apply for safe, limited cases.

## Out of scope
- Real-money execution.
- Opaque auto-switching.
- RL / autonomous optimizer.
- Enterprise workflow complexity.
