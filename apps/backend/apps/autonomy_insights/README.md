# autonomy_insights

Cross-campaign lessons registry and governance learning synthesis for autonomy lifecycle artifacts.

## Responsibilities
- Consume closed lifecycle evidence from `autonomy_closeout`, `autonomy_followup`, and `autonomy_feedback`.
- Generate auditable `CampaignInsight` and `InsightRecommendation` records.
- Provide manual-first review APIs and summary metrics for cockpit/UI integration.

## Explicit boundaries
- Recommendation-first only: no automatic roadmap/scenario/program/manager apply.
- No real-money or live broker/exchange execution.
- No opaque ML/LLM authority; rule-based deterministic synthesis only.
